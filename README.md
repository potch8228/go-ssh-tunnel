Tunnel - A quick ssh tunneling utility
======================================

### This is still work in progress... ###

Usage
=====
./tunnel [args]

Args
====
-ssh  
-remote  
-local  
-user  
-pwd  
-key (in PEM format) 

Dependency
==========
golang.org/x/crypto/ssh

Thanks
======
